nodeGraph::nodeGraph ( QGraphicsScene *  scene,
MainWindow *  parent = 0 
)

Main nodegraph class that performs operations on nodes.

All operations that add, move, connect or delete nodes are performed here, also holds the view into scne that holds the DAG objects. Subclasses QGraphicsView.

Parameters
sceneScene that holds graphical representation of DAG
parentMain UI window
